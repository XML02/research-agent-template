# Starter Checklist for New Research Projects

Use this checklist when starting a new project from this template.

## Setup
- [ ] Clone repository
- [ ] Install dependencies: `uv sync` or `pip install -r requirements.txt`
- [ ] Run initial test: `pytest -v`
- [ ] Set up data directories: `data/raw/`, `data/output/`

## First Issue
- [ ] Create issue from template
- [ ] Assign to yourself
- [ ] Write handoff prompt
- [ ] Run agent
- [ ] Review PR
- [ ] Merge

## Style Guide Alignment
- [ ] Read relevant section of Genskow & Shapiro style guide
- [ ] Note any project-specific conventions
- [ ] Add to this checklist if needed

## AI Use Disclosure
- [ ] Every AI-assisted file includes disclosure in docstring
- [ ] Commit messages note AI use: `[AI: tool used]`
- [ ] Final manuscript includes AI disclosure section

## Pre-Submission
- [ ] All pytest tests pass
- [ ] All issue acceptance criteria met
- [ ] AI disclosure statement drafted
- [ ] Co-authors have reviewed AI-generated content
